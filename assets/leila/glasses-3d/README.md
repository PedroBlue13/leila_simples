# Óculos 3D interativos

Versão isolada da cena pública **Glasses - Interactive Hero Concept**, criada por Jordan (`@jlindz`).

Fonte:
<https://app.spline.design/community/file/9337a696-c43f-41ab-ac4a-6c490eeaf318>

Foram ocultados o fundo, os textos e a navegação da cena original. A armação 3D e o comportamento de acompanhar o cursor foram preservados.

O runtime, o WebAssembly e a textura de iluminação estão incorporados no `index.html`, permitindo usar a cena mesmo quando o site principal é aberto diretamente por `file://`.
